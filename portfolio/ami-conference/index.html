UX / UI Design

AMI Conference 2025

A web and visual design project for the AMI International Monetary Reform Conference 2025. Responsible for designing the event webpage and promotional materials, including posters and social media graphics — focusing on clear information hierarchy, accessibility, and consistent visual identity across digital formats.

About this project

This project involved designing both the event webpage and a set of promotional materials for the AMI International Monetary Reform Conference 2025. The goal was to communicate complex economic topics in a way that is clear, structured, and visually engaging, while maintaining a professional and trustworthy tone aligned with the organisation’s identity.

The webpage design served as the central information hub for the conference, presenting key details such as speakers, schedule, topics, and registration in a structured and user-friendly layout. In parallel, supporting assets such as posters and social media graphics were created to promote the event and drive traffic to the webpage.

A key challenge was handling high information density. The content includes detailed descriptions of economic topics, multiple speakers, and logistical information. To address this, the design focuses heavily on visual hierarchy, grouping, and progressive disclosure, allowing users to quickly scan the page while still accessing deeper information when needed.

My contributions included:

  • Designing the full webpage layout and structure (UI/UX).
  • Creating promotional assets (posters and social media graphics).
  • Structuring complex content into clear, digestible sections.
  • Establishing a consistent visual system across web and marketing materials.
  • Designing reusable layout patterns for speakers, topics, and event information.
  • Ensuring readability and accessibility across devices.

The final result is a cohesive design system where the webpage and promotional materials reinforce each other, improving both usability and communication.

Design comments
Conference Webpage Design

This design serves as the main information hub for the event. It demonstrates how complex content can be structured into clear sections, with strong visual hierarchy and reusable components such as speaker profiles. The layout balances readability and visual engagement, guiding users from high-level information to specific details and actions.

Conference Posters

The posters presents a condensed version of the webpage content, focusing on key information such as topics, speakers, and event details. The design uses hierarchy and spacing to maintain readability despite the high information density, while staying visually consistent with the webpage.

Keynote Speaker Social Media Graphics

These graphic highlight individual speakers using a simplified and focused layout. The large portrait and clear typography create a strong focal point, while maintaining consistency with the overall visual system. The design is optimized for quick engagement on social media platforms.
